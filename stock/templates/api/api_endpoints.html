Endpoints de la API

  • GET /api/items/ - Obtener una lista de todos los artículos.
  • POST /api/items/ - Crear un nuevo artículo. (Requiere datos del artículo en el cuerpo de la solicitud)
  • GET /api/items/<id>/ - Obtener los detalles de un artículo específico.
  • PUT /api/items/<id>/ - Actualizar un artículo específico. (Requiere datos actualizados en el cuerpo de la solicitud)
  • DELETE /api/items/<id>/ - Eliminar un artículo específico.
